# FAQ

## Why does login fail with MongoDB server selection timeout?

Most often the Atlas cluster is paused or your IP is not allowed.

- Resume cluster in Atlas
- Add your current IP (or 0.0.0.0/0 for local development)
- Restart the dev server after env updates

## Why do I see Next.js missing module errors from .next/server?

This is a dev cache/process issue.

Run:

```bash
pkill -f "next dev --port 3000" || true
rm -rf .next
npm run dev
```

Also keep only one Next.js dev process active.

## Can mobile browser camera be used for monitoring?

Yes. Admin camera flow supports browser/mobile camera mode via getUserMedia where the browser permits camera access.

## Which endpoint starts monitoring?

Use same-origin proxy endpoint:

- POST /api/monitor/start

And stop with:

- POST /api/monitor/stop

## Where can I check monitor status?

- GET /api/monitor/status/:classId
- GET /api/monitor/active

## How do I export attendance data?

Call:

- GET /api/attendance?format=csv

You can combine with filters such as classId, studentId, startDate, and endDate.

## Does the app support multi-tenant institutions?

Yes. Core data routes use institution scoping through tenant context resolution.

## What Node version should I use?

Node 22 LTS is recommended for stable local development with the current Next.js setup.

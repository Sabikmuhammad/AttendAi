# Installation

## Prerequisites

- Node.js 20+ (22 LTS recommended)
- Python 3.8+
- MongoDB Atlas connection string
- Clerk keys

## 1) Frontend setup

```bash
npm install
```

Create .env.local in project root:

```env
NEXT_PUBLIC_CLERK_PUBLISHABLE_KEY=...
CLERK_SECRET_KEY=...
MONGODB_URI=mongodb+srv://<user>:<password>@<cluster>.mongodb.net/?appName=Cluster0
NEXT_PUBLIC_AI_SERVICE_URL=http://localhost:8000
AI_SERVICE_URL=http://localhost:8000
PYTHON_SERVICE_URL=http://localhost:8000
```

## 2) AI service setup

```bash
cd ai-service
python3 -m venv venv
source venv/bin/activate
pip install -r requirements.txt
```

Create ai-service/.env:

```env
MONGODB_URI=mongodb+srv://<user>:<password>@<cluster>.mongodb.net/?appName=Cluster0
DATABASE_NAME=test
```

## 3) Start both services

Terminal 1:

```bash
cd ai-service
source venv/bin/activate
python main.py
```

Terminal 2:

```bash
npm run dev
```

## 4) Verify

- Web app: http://localhost:3000
- AI service: http://localhost:8000
- Docs: http://localhost:3000/docs

## Troubleshooting quick checks

### MongoDB timeout during login

- Resume paused Atlas cluster
- Add your IP in Atlas Network Access
- Validate MONGODB_URI credentials and encoding

### Next.js missing chunk/module in dev

```bash
pkill -f "next dev --port 3000" || true
rm -rf .next
npm run dev
```

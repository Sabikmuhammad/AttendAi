# AttendAI Documentation

AttendAI is a multi-tenant classroom attendance platform with:

- Next.js web app for admin, faculty, and student roles
- MongoDB + Mongoose data layer
- FastAPI-based AI service for face detection and monitoring
- Cookie-based auth with institution scoping

## What this documentation covers

- Local setup for frontend and AI service
- Current route structure and role flows
- Real API endpoints available in this repository
- Browser camera and AI monitoring workflow
- MongoDB and runtime troubleshooting

## Current core app areas

- Admin: dashboard, students, faculty, classes, camera, profile
- Faculty: dashboard, create class, live class
- Student: dashboard and attendance views
- Auth and onboarding: login, register, OTP verification, role setup

## Important runtime notes

- MongoDB Atlas free clusters can pause. If login fails with server selection timeout, resume the cluster and check Network Access.
- If Next.js dev throws missing module errors from .next/server, stop all dev processes, clear .next, then restart.
